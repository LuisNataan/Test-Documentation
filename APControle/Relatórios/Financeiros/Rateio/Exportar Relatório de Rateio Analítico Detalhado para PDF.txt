Exportar relatório de rateio analítico detalhado para PDF
Pré-requisitos:
- Deve existir lançamentos financeiros cadastrados no sistema
- Deve existir um período definido para o relatório
- Acessar Relatórios -> Financeiros -> Rateio -> Rateio Analítico Detalhado

Dado que estou na tela de Relatórios de Rateio Analítico Detalhado
E seleciono um condomínio
E seleciono o período desejado para o relatório
E clico no botão Exportar para PDF
Quando o sistema exporta o relatório de rateio detalhado para PDF
Então o relatório deve ser exportado para o formato PDF
